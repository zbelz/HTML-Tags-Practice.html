HTML Tags Cheat Sheet

Headings

Headings from 'h1' to 'h6' identify blocks and sections. Every pages must have one 'h1'. (This is important because google searches show you 'h1' line and Not having an 'h1' line will derank you in the searchs.)

Paragraphs

Paragraphs of text go in the 'p' tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in 'ul' tags. Each list item goes in an 'li' tag.

Types of lists

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered Lists

Ordered lists are numbers and go in 'ol' tags. Each list item goes in an 'li' tag.

Anchors (i. e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go to DuckDuckGo to search for more information.

Setting 'target="_blank"' will open the link in a new window. See MDN for more details.

Text Emphassis

Italic

The 'em' element adds exra emphasis to the text within the openening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the 'strong' element to emphasize text. This wil show up as bold letters in most browsers.
